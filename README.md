# md2hiki

md2hiki is a markup converter, from Markdown to Hiki(https://github.com/hiki/hikidoc) format.

## Installation

Add this line to your application's Gemfile:

    gem 'md2hiki'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install md2hiki

## Usage

    $ md2hiki foo.md > foo.hiki
